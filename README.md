### Instalación

Para ejecutar el proyecto:

   
 #### Instalar dependencias de Laravel y PHP:

```bash
   composer install
```

 #### Configurar variables de entorno:
 
```bash
   cp .env.example .env
   php artisan key:generate
```

 #### Instalar dependencias de frontend (SASS para animaciones de la intro de la aventura):

 ```bash
   npm install
   npm install -D sass-embedded
```
 #### Ejecutar aplicación:
 
```bash
   php artisan serve
   npm run dev
```
